# Ekatraa Web – Landing Page

Modern Next.js landing page for **Ekatraa Mobile App** and **Ekatraa Vendor App**, based on the Ekatraa brand and presentation content.

## Features

- **Next.js 14** (App Router) with TypeScript
- **Tailwind CSS** with Ekatraa brand colors (#FF4117, #FF7700, #918F8F)
- **Framer Motion** for scroll and entrance animations
- **Responsive** layout for all screen sizes
- **Sections**: Hero, About, Our Apps (Mobile + Vendor), Features, USP, CTA, Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project Structure

- `src/app/` – Next.js App Router (layout, page, globals.css)
- `src/components/` – Nav, Hero, About, AppsSection, Features, USP, CTA, Footer
- `public/` – Ekatraa logo (ekatraa.svg)

## Brand

- **Primary**: #FF4117 (Ekatraa red-orange)
- **Gradient**: #FF0000 → #FF7700
- **Accent**: #918F8F (gray)
